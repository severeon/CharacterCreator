# SAVAGE SPECIES

*3.5 Edition Rules Reference*

*A complete rules guide for playing monster characters using 3.5 Edition
mechanics*

This document is a fan-made conversion reference, not a reproduction of
copyrighted material.

## SECTION 1: CORE CONCEPTS FOR MONSTER CHARACTERS

Playing a monster as a player character follows the same basic framework
as a standard character, with additional considerations for racial Hit
Dice, level adjustments, and monster class progression. All rules in
this reference use 3.5 Edition mechanics.

### 1.1 Key Terms

  -----------------------------------------------------------------------
  **Term**              **Definition**
  --------------------- -------------------------------------------------
  Racial Hit Dice (RHD) Hit Dice a creature possesses by virtue of its
                        monster type, before any class levels. These
                        function like class levels for saves, BAB, and
                        skills.

  Level Adjustment (LA) A penalty added to a monster\'s effective level
                        to account for powerful racial abilities. LA does
                        not grant additional HP, skill points, or class
                        features.

  Effective Character   ECL = Racial HD + Level Adjustment + Class
  Level (ECL)           Levels. This is used to determine XP thresholds
                        and party balance.

  Monster Class         A special class progression from Savage Species
                        that lets a player \'grow into\' a monster over
                        multiple levels, beginning at ECL 1.

  Challenge Rating (CR) CR measures NPC/encounter difficulty. It is NOT
                        the same as ECL and should not be used to set LA.
  -----------------------------------------------------------------------

**1.2 The ECL Formula**

**ECL = Racial Hit Dice + Level Adjustment + Class Levels**

Example: A minotaur has 6 RHD and LA +2. A minotaur fighter 2 has ECL 10
(6 + 2 + 2). The character is treated as a 10th-level character for XP
and treasure purposes but has the class features of a 2nd-level fighter.

> *CR is not ECL. A minotaur has CR 4 but ECL 8 (before class levels).
> Using CR to set ECL will result in unbalanced characters.*

**1.3 When to Use Monster Classes**

Monster classes are the preferred method when a player wants to begin
play at ECL 1 with a monster race that would otherwise require starting
at a higher ECL. The monster class parcels out the creature\'s abilities
gradually across multiple levels, allowing the character to participate
in a low-level campaign.

If the campaign begins at a higher ECL (equal to or exceeding the
monster\'s RHD + LA), simply apply the full monster stats and let the
player begin taking class levels immediately.

**SECTION 2: MONSTER CHARACTER CREATION (3.5 RULES)**

**Step 0: Check With Your DM**

Before building a monster character, confirm the following with your DM:

- Is this monster type allowed in the campaign?

- Will the monster class progression be used, or will the character
  start at full ECL?

- How will XP penalties for multiclassing interact with monster class
  levels?

- Are any monster abilities being adjusted for campaign balance?

**Step 1: Choose Your Monster Kind**

Select a monster from the 3.5 Monster Manual. The monster\'s entry
defines your base racial statistics. If the monster appears in Savage
Species but has been updated in the 3.5 MM, always use the 3.5 MM
statistics as the authoritative source.

**Step 2: Determine Racial Hit Dice**

A monster\'s Hit Dice determine its base attack bonus, base saving
throws, and number of skill points before class levels are applied.

Base Attack Bonus by creature type (3.5 rules):

  ------------------------------------------------------------------------
  **Creature Type**     **BAB Progression**   **Good Saves**
  --------------------- --------------------- ----------------------------
  Aberration            3/4 HD (as Cleric)    Will

  Animal                3/4 HD (as Cleric)    Fort, Ref

  Construct             3/4 HD (as Cleric)    None (immune)

  Dragon                Full HD (as Fighter)  Fort, Ref, Will

  Elemental             3/4 HD (as Cleric)    Ref or Fort (by type)

  Fey                   1/2 HD (as Wizard)    Ref, Will

  Giant                 3/4 HD (as Cleric)    Fort

  Humanoid              3/4 HD (as Cleric)    Varies by subtype

  Magical Beast         Full HD (as Fighter)  Fort, Ref

  Monstrous Humanoid    Full HD (as Fighter)  Ref, Will

  Ooze                  3/4 HD (as Cleric)    None

  Outsider              Full HD (as Fighter)  Fort, Ref, Will

  Plant                 3/4 HD (as Cleric)    Fort

  Undead                1/2 HD (as Wizard)    Will

  Vermin                3/4 HD (as Cleric)    Fort
  ------------------------------------------------------------------------

**Step 3: Determine Level Adjustment**

Level adjustment is assigned based on the monster\'s overall power
relative to a standard PC race. Use the 3.5 Monster Manual LA listing as
the primary source. If a monster has no LA listed in the 3.5 MM, it is
not considered suitable for PC play without DM approval.

General LA guidelines:

- Powerful ability score bonuses above +4 net add LA

- High natural armor bonuses add LA

- Flight, invisibility, and other highly impactful abilities add LA

- Spell-like abilities available multiple times per day add LA

- Large size alone does not typically add LA

- Immunities add small LA; extraordinary senses (darkvision, scent) add
  little to none

> *If the 3.5 MM lists a different LA than Savage Species, always use
> the 3.5 MM value.*

**Step 4: Calculate ECL and Starting Level**

To use a monster in a campaign starting at ECL 1, use the monster class
progression from Savage Species with the following adjustments for 3.5
rules (see Section 3).

To start at full power, the campaign must begin at an ECL equal to at
least (RHD + LA). The player then adds class levels from there.

**Step 5: Ability Scores**

Two methods are available:

**Method A: Point Buy / Standard Array**

Generate ability scores normally, then apply the monster\'s racial
ability score modifiers as listed in the 3.5 MM. These modifiers
represent the net adjustment from average (10/11).

**Method B: Elite Array from Monster Stats**

Take the monster\'s base ability scores from the 3.5 MM entry and use
them directly. This represents a typical member of the species. Scores
should reflect the 3.5 MM entry, not the Savage Species entry, if they
differ.

> *Ability score increases from monster class levels: Monster PCs gain
> +1 to any ability score every 4 character levels (counting all levels,
> including monster class levels), just like standard PCs. This is a
> deliberate PC advantage over NPC versions of the same creature.*

**Step 6: Skills**

Monster PCs gain skill points based on their creature type\'s skill
point allocation and Intelligence modifier. Use 3.5 Edition skill rules
throughout. Several skills changed between 3.0 and 3.5 --- see Section 5
for the full skill conversion table.

First-level skill points: Multiply the per-level total by 4 (for the
first level of the monster class or first character class level,
whichever comes first).

Class skills for each creature type are listed in the 3.5 MM
introduction. Cross-reference the specific monster\'s entry for its
skill list.

**Step 7: Feats**

A monster with RHD gains feats at the normal rate: one feat at 1 HD, and
one additional feat every 3 HD thereafter (at HD 1, 3, 6, 9, etc.).

Monster class feats are granted as listed in the monster class table.
For feat name discrepancies between 3.0 and 3.5, see Section 6.

**Step 8: Special Qualities and Special Attacks**

Apply all special qualities and special attacks as listed in the 3.5 MM
for the creature. When the Savage Species monster class table grants an
ability that has changed in 3.5, use the 3.5 version of that ability.

Key areas requiring updates:

- Damage Reduction: Entirely new system (see Section 3)

- Energy Resistance: Changes from per-round to per-hit (see Section 4)

- Shapechanger: Now a subtype, not a type (see Section 7)

- Spell-like abilities: Use 3.5 spell descriptions (see Section 8)

**3.4 Combining DR**

When a character has DR from multiple sources (monster class +
template + magic item), only the highest DR value applies unless both
have the same bypass condition. DR does not stack by default. The DM may
rule otherwise for narrative reasons.

**SECTION 4: ENERGY RESISTANCE (3.5 CONVERSION)**

**4.1 Per-Round vs. Per-Hit**

In 3.0 Edition, energy resistance was measured per round: a creature
with Fire Resistance 20 would absorb up to 20 points of fire damage
total each round regardless of how many fire attacks hit it.

In 3.5 Edition, energy resistance is measured per hit: a creature with
Resist Fire 20 subtracts 20 from each individual fire attack\'s damage.

**4.2 Conversion Guidelines**

Because per-hit resistance is weaker against multiple rapid attacks than
per-round resistance was, reduce the numeric value slightly when
converting:

  -----------------------------------------------------------------------
  **3.0 Per-Round          **Suggested 3.5 Per-Hit Resistance**
  Resistance**             
  ------------------------ ----------------------------------------------
  Resist 5/round           Resist 5 per hit (unchanged; single-hit
                           difference is negligible)

  Resist 10/round          Resist 10 per hit (approximately equivalent
                           for most encounters)

  Resist 20/round          Resist 12--15 per hit (reduce to account for
                           multi-hit spells)

  Resist 30/round          Resist 20 per hit

  Complete immunity        Complete immunity (no change)
  -----------------------------------------------------------------------

> *The reduction matters most when facing spells or attacks that deal
> damage in multiple hits per round (e.g., Magic Missile, multiple
> natural attacks with energy damage). For single-hit fire/cold spells,
> the per-hit value can remain roughly equivalent to the per-round
> value.*

**7.2 Electricity Subtype**

The Electricity subtype was eliminated in 3.5. Creatures that had it in
3.0 should instead have Immunity to Electricity listed under Special
Qualities.

**7.3 Cold and Fire Subtypes**

These subtypes received clearer mechanics in 3.5. A creature with the
Fire subtype takes half again as much fire damage (+50%) regardless of
whether a saving throw is allowed, and is immune to fire. The same
applies to Cold. This is automatically factored in --- no conversion is
required unless a Savage Species entry described different mechanics.

**8.1 Conjuration and Spell Resistance**

In 3.5, conjuration spells (including most summoning and calling spells)
generally do not allow Spell Resistance checks by default, even if they
deal damage. This is a change from 3.0. When a monster has a SLA that is
a conjuration, it bypasses SR unless the spell description specifically
notes otherwise.

**SECTION 9: MONSTER CLASS RULES (3.5 EDITION)**

**9.1 What Monster Classes Do**

A monster class is a special character class that parcels out a
monster\'s racial abilities across multiple levels. This allows a player
to start at ECL 1 as a weak version of a powerful monster, gaining new
abilities and stat increases as they level up, until the class is
complete and the character matches the full monster\'s statistics.

Once the monster class is complete, the player may begin taking levels
in standard character classes. The completed monster class is treated as
the character\'s racial identity, not as character class levels for
multiclassing penalty purposes.

**9.2 Monster Class Progression Structure**

Each monster class table specifies the following per level:

- Hit Die type (e.g., d8, d10, d12)

- Base Attack Bonus progression

- Good and poor saving throw progressions

- Skill points per level

- Feats gained at specific levels

- Special abilities gained (SLAs, natural attacks, special qualities,
  ability score increases, natural armor increases)

- Level Adjustment notation

**9.3 Multiclassing with Monster Classes (3.5 Rule Update)**

Savage Species (3.0) required that once a player began a monster class,
they had to complete it before taking any other class levels. This
restriction does not apply to the Savage Progressions web articles (the
3.5-era equivalent), and many DMs house-rule it away for Savage Species
as well.

Recommended 3.5 approach:

- Characters are not required to complete monster class levels in
  uninterrupted succession

- A character may take a level of a standard class, then return to the
  monster class at a later level

- The first level of the monster class must still be taken before the
  second, and so on

- XP penalties for multiclassing apply normally using ECL, not monster
  class level count

> *If your DM uses the strict Savage Species rule (complete the class
> before any other class), note that this can create significant balance
> problems for spellcasting monster classes at higher ECLs. Discuss with
> your DM before committing to this interpretation.*

**9.4 Ability Score Increases**

Monster PCs gain ability score increases at every 4th character level
(ECL 4, 8, 12, 16, 20), just like standard PCs. This counts all levels
--- monster class levels and character class levels combined.

This is intentional: it gives monster PCs a slight advantage over NPC
versions of the same creature and compensates partially for the ECL cost
of the level adjustment.

**9.5 Hit Points During Monster Class Progression**

At each monster class level, the character rolls the class\'s Hit Die
and adds their Constitution modifier (or takes the average die result,
if the DM allows that variant). This is identical to rolling HP for any
character class level.

**9.6 Skills During Monster Class Progression**

The monster class lists skill points per level. Use the 3.5 skill list
(see Section 5 for conversions). Class skills during monster class
progression are those listed for the creature type in the 3.5 MM, plus
any additional class skills listed in the monster class entry.

First level: Multiply skill points by 4, as with any first level.

**9.7 Natural Attacks**

Monster natural attacks use the 3.5 rules for natural weapons:

- A creature making only natural attacks in a round may make a full
  attack with all of them at no penalty

- If a creature also wields a manufactured weapon, its secondary natural
  attacks take a -5 penalty

- Primary natural attacks use full BAB; secondary natural attacks use
  BAB -5

- Natural weapons are treated as magic weapons for the purpose of
  overcoming DR if the creature has 5 or more HD (3.5 rule)

> *The 3.5 rule that natural weapons count as magic for DR purposes at
> 5+ HD does not apply to the DR of other creatures\' natural weapons
> --- it applies to the attacking creature\'s natural weapons when
> determining if they overcome the DR of the target.*

**9.8 Templates and Monster Classes**

Some Savage Species entries include templates that can be applied to
monster characters. When applying a template from Savage Species to a
3.5 character:

- Use the 3.5 MM version of the template if one exists

- Convert all DR using Section 3 of this document

- Convert all energy resistance using Section 4

- Update creature type if the template changes it (check 3.5 MM)

- Level adjustment from the template stacks with level adjustment from
  the base race

**SECTION 11: PRESTIGE CLASSES FROM SAVAGE SPECIES (3.5 NOTES)**

Savage Species includes several prestige classes. Most function fine in
3.5 with minor skill name updates (see Section 5). The following notes
apply:

- Any prestige class that has a BAB prerequisite: use the 3.5 BAB
  progression for the creature\'s type when determining if the monster
  qualifies

- Skill prerequisites: substitute 3.5 skill names using the conversion

**SECTION 13: SAVAGE PROGRESSIONS --- THE 3.5 ALTERNATIVE**

**13.1 What Are Savage Progressions?**

Savage Progressions was a series of free web articles published by
Wizards of the Coast after the release of 3.5 Edition. They provide
monster class progressions for popular playable races using native 3.5
rules --- meaning they require no conversion. They were inspired by the
Savage Species framework but were written from the ground up for 3.5.

**13.2 Key Rule Differences from Savage Species**

**13.3 Which Races Have Savage Progressions?**

The following races received Savage Progressions articles. If you want
to play one of these, use the Savage Progressions version instead of
Savage Species:

- Aasimar

- Bugbear

- Drow

- Gnoll

- Half-dragon (several variants)

- Half-fiend

- Hobgoblin

- Kobold

- Lizardfolk

- Minotaur

- Ogre

- Tiefling

- Troll

- Yuan-ti Pureblood

> *Savage Progressions articles were hosted on the Wizards of the Coast
> website. They have been archived online. Search for \'D&D Savage
> Progressions web archive\' to find them.*

**13.4 Using Both Together**

There is no conflict in using Savage Progressions for some characters
and Savage Species (converted) for others in the same campaign. Apply
the Savage Progressions multiclassing rule to Savage Species characters
as a house rule for consistency.

**APPENDIX A: QUICK CONVERSION CHECKLIST**

Use this checklist every time you convert a monster class entry from
Savage Species to 3.5 rules:

1.  Damage Reduction: Convert DR X/+Y to DR X/\[type\] using Section 3.
    Reduce the numeric value as needed.

2.  Energy Resistance: Change from per-round to per-hit. Adjust value if
    needed (Section 4).

3.  Skills: Replace all 3.0 skill names with 3.5 equivalents (Section
    5).

4.  Feats: Replace renamed or restructured feats with 3.5 versions
    (Section 6).

5.  Creature Type: If Shapechanger was the type, assign a primary type +
    Shapechanger subtype (Section 7).

6.  Type Traits: Apply 3.5 automatic type traits (Section 7). Remove any
    3.0-specific trait descriptions that contradict them.

7.  Spell-Like Abilities: Use 3.5 spell descriptions. Key spells: Haste,
    Polymorph, stat buffs (Section 8).

8.  Level Adjustment: Check the 3.5 MM entry. If LA differs, use 3.5 MM
    value.

9.  Stat Block Endpoint: Verify the monster class ends at stats matching
    the 3.5 MM entry, not the 3.0 entry.

10. Check Section 12: If your monster is on the significant-changes
    list, do additional review.

11. Check Section 13: If a Savage Progressions article exists for your
    monster, use it instead.
